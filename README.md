# TraktTV REST API Integration

## Overview

This project demonstrates integration with the TraktTV REST API using Python. The application authenticates with the API, retrieves movie data, and performs operations such as adding movies to a watchlist, marking them as watched, and managing user data.

## Technologies

- Python
- Requests
- REST API
- JSON
- TraktTV API

## Features

- Authentication using API credentials and access token
- Retrieve popular movies
- Fetch movie details
- Add movies to a watchlist
- Update watchlist items
- Mark movies as watched
- Delete data using API requests

## HTTP Methods Used

- **GET** – retrieve movie information
- **POST** – add movies to a watchlist
- **PATCH/POST** – update watchlist and viewing history
- **DELETE** – remove data
